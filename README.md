# NidoInformativoDeMigrantesCDMX

## GitHub Secret: MAPBOX_TOKEN

This project uses a GitHub Actions workflow to generate `js/secrets.js` from a repository secret.

1. Go to your repository settings.
2. Add a new secret named `MAPBOX_TOKEN`.
3. Push to `main` to trigger the workflow.

For local development, copy `js/secrets.example.js` to `js/secrets.js` and replace the placeholder token.
